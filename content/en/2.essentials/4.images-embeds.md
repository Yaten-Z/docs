## Markdown

使用标准 Markdown 语法显示图片或视频。

### 图片

::code-preview
![Nuxt 社交图片](https://nuxt.com/new-social.jpg)

#code
```mdc
![Nuxt 社交图片](https://nuxt.com/new-social.jpg)
```
::

也可以使用本地图片：

::code-preview
![日落时分，云海之上的雪山](/mountains.webp)

#code
```mdc
![日落时分，云海之上的雪山](/mountains.webp)
```
::

::note{to="https://image.nuxt.com/"}
Docus 在底层会使用 `<NuxtImg>` 组件，而非原生的 `img` 标签。
::
